# 📚 משחק אוצר מילים באנגלית

אפליקציית Streamlit ללימוד אוצר מילים באנגלית — מותאמת לתלמידות ותלמידים בכיתה ה', עם ממשק בעברית, משחקים, סיפורים, למידה מותאמת אישית ומעקב התקדמות.

## ✨ מה יש באפליקציה?

| חלק | תיאור |
|-----|--------|
| 🌟 **משימה יומית** | 3 מילים חדשות + 3 קשות + 2 חזרה, תרגול, סיפור — כל יום |
| 📚 **סיפורים** | סיפור מותאם אישית — מילים קשות, מילות מורה ומבחן |
| 📖 **סיפורים שלמדתי** | היסטוריית סיפורים עם ניקוד ומילים |
| 📖 **מילים** | מילים באנגלית עם תרגום, משפט לדוגמה, וכפתור 🔊 להגייה |
| 🎮 **תרגול** | שאלות חכמות — מילים קשות ומילים למבחן קודם, עם ערבוב מילים קלות |
| 📚 **סיפור** | סיפור קצר עם מילים מודגשות, 🔊 לסיפור ולמילים, 2–5 שאלות הבנה |
| 🧠 **הלמידה שלי** | מילים קשות, מילים שכבר יודעים, וסיכום להורים |
| 📊 **התקדמות** | שמירה מקומית ב-SQLite — תשובות נכונות ושגויות |
| 👩‍🏫 **מצב מורה** | הוספת מילים, ניהול תלמידות (סיסמה: **`teacher123`**) |
| 👧 **תלמידות** | כניסה עם סיסמה אישית — התקדמות נפרדת לכל תלמידה |

## 🔐 סיסמאות

| סוג | סיסמה | הערות |
|-----|--------|--------|
| **מורה** | `teacher123` | כניסה למצב מורה — הוספת מילים וניהול תלמידות |
| **תלמידה** | `שם123` | ברירת מחדל: שם + `123` (למשל: **נועה** → `נועה123`) |

סיסמות נשמרות **מקומית** ב-`progress.db`. ניתן לשנות סיסמת תלמידה במצב מורה.

## 👧 פרופילים אישיים לכל תלמידה

כל תלמידה נוצרת **רק במצב מורה** (לא מהמסך הראשי), עם:

- **שם** (למשל: נועה)
- **כיתה** (ב' – ו')
- **אווטאר** 🦄 🐱 🐶 🐼 🦊 🐰 🐨 🐯
- **סיסמה** (ברירת מחדל: שם + `123`)

### כניסה לתלמידה

1. בהפעלה — מסך **"מי לומדת היום?"** בלבד (ללא שיעורים/תרגול).
2. אם אין תלמידות: **"אין עדיין תלמידות. היכנסי למצב מורה כדי להוסיף תלמידה."**
3. בחרי כרטיס תלמידה → הכניסי **סיסמה**.
4. אחרי כניסה מוצלחת: **"🦄 שלום נועה! מוכנה ללמוד אנגלית?"**
5. בתפריט הצד: **🔄 החלפת תלמידה** — יציאה וחזרה למסך הכניסה.

### ניהול תלמידות (מצב מורה)

1. בתפריט הצד (לפני כניסת תלמידה) או אחרי כניסה — **👩‍🏫 מצב מורה**.
2. הכניסי סיסמת מורה: **`teacher123`**
3. אפשרויות:
   - **➕ הוספת תלמידה** — שם, כיתה, אווטאר, סיסמה (ריק = שם123)
   - **✏️ עריכת תלמידה** — שינוי שם, כיתה, אווטאר, סיסמה
   - **🗑️ מחיקת תלמידה** — מוחק גם את כל ההתקדמות

**ללא כניסת תלמידה** — לא ניתן לתרגל. מצב מורה זמין תמיד.

כל תלמידה שומרת **בנפרד**: מילים קשות, סטטיסטיקות, היסטוריית תרגול וציוני קושי.

**כיתה:** נשמרת לכל תלמידה. שיעור ברירת המחדל נבחר לפי כיתה (ב'→1, ג'→2, ד'→3, ה'/ו'→4) — ניתן לשנות ידנית.


## 🌟 המשימה היומית שלי

הזרימה הראשית של האפליקציה — לא צריך לבחור שיעור ידנית.

### מה כוללת משימה יומית?

| חלק | תוכן |
|-----|------|
| ✨ **מילים חדשות** | 3 מילים מתאימות לכיתה (מילות מורה/מבחן קודם!) |
| 😅 **מילים קשות** | 3 מילים מההיסטוריה האישית |
| 🔄 **חזרה** | 2 מילים שכבר למדת |
| 🎮 **תרגול** | שאלות בחירה + השלמת משפטים |
| 📚 **סיפור** | סיפור עם מילות היום מודגשות, 🔊 לסיפור ולמילים, 2–5 שאלות הבנה |

### סיפורים מותאמים אישית

| תכונה | תיאור |
|--------|--------|
| 📚 **סיפורים** | סיפור חדש לפי המילים שלך — מילים קשות, מילות מורה ומבחן |
| 🔊 **השמעה** | כפתור 🔊 לכל הסיפור ולכל מילת אוצר בסיפור |
| ✨ **הדגשה** | מילות האוצר מודגשות בתוך הסיפור |
| ❓ **שאלות** | 2–5 שאלות הבנה — נשמרות לכל תלמידה |
| 📖 **היסטוריה** | **סיפורים שלמדתי** — כל הסיפורים עם ניקוד ותאריך |
| 🔄 **חזרה** | מילים קשות מופיעות יותר בסיפורים הבאים |

### סיום יומי

- האפליקציה **זוכרת** אם סיימת את המשימה היום.
- אחרי סיום: **"סיימת את המשימה היומית 🎉"**
- אפשרויות נוספות:
  - **תרגול נוסף**
  - **חזרה על מילים שקשה לי**

### עדיפות למילות מורה

מילים שהמורה הוסיפה — במיוחד **למבחן** 📝 — מקבלות עדיפות גבוהה במשימה היומית.

### כל המילים

בתפריט **📖 כל המילים** אפשר לעיין בכל אוצר המילים. סינון לפי שיעור (1–4) נמצא בתוך כרטיסיה מתקפלת — לא בתפריט הראשי.

## 🧠 למידה מותאמת אישית (Spaced Repetition)

האפליקציה **זוכרת** לכל מילה:

- כמה פעמים נראתה (`times_seen`)
- כמה פעמים נענתה נכון / שגוי
- מתי נראתה לאחרונה
- **ציון קושי** (`difficulty_score`) — 0 עד 100

### איך זה עובד?

| אירוע | השפעה |
|--------|--------|
| מילה חדשה | קושי בינוני (50) |
| תשובה נכונה | הקושי יורד |
| תשובה שגויה | הקושי עולה |
| מילת מורה / למבחן | עדיפות גבוהה יותר בתרגול |

### בחירת מילים לתרגול

האפליקציה **לא** בוחרת מילים באקראי:

1. קודם מילים **למבחן** 📝 ומילות **מורה** ⭐
2. אחר כך מילים עם **קושי גבוה**
3. מערבבת **1–2 מילים קלות** כדי שלא יהיה מתסכל

### מסך "הלמידה שלי"

- **😅 מילים שקשה לי** — המילים הקשות ביותר, עם תרגום, נכון/שגוי, ו-🔊
- **🌟 מילים שאני כבר יודעת** — מילים עם הצלחה גבוהה (לפחות 3 ניסיונות, 75%+ נכון)
- **👨‍👩‍👧 סיכום להורים** — כמה מילים תורגלו, חזקות, קשות, והמלצה בעברית

## 📁 מבנה הקבצים

```
english_game/
├── app.py           # האפליקציה הראשית
├── database.py      # ניהול מסד נתונים SQLite
├── words.csv        # אוצר המילים (4 שיעורים × 5 מילים)
├── stories.csv      # סיפורים ושאלות הבנה
├── requirements.txt # תלויות Python
├── .cache/tts/      # קבצי הגייה (נוצר אוטומטית)
└── README.md        # הקובץ הזה
```

לאחר הרצה ראשונה נוצר אוטומטית הקובץ `progress.db` עם:

- נתוני תרגול
- מילים שהמורה הוסיפה
- **ביצועים לכל מילה** (`word_performance`)
- **משימות יומיות** (`daily_sessions`) — מילים וסטטוס השלמה לכל יום
- **סיפורים** (`student_stories`) — היסטוריית סיפורים, ניקוד ומילים לכל תלמידה

**הערה:** השמעת הגייה (gTTS) דורשת חיבור לאינטרנט **רק בפעם הראשונה** לכל מילה/טקסט.

## 🚀 איך להריץ מקומית?

```bash
cd english_game
python3 -m venv venv
source venv/bin/activate   # macOS / Linux
pip install -r requirements.txt
streamlit run app.py
```

פתחו בדפדפן: **http://localhost:8501**

## 🎮 איך משתמשים?

1. **היכנסי** — בחרי תלמידה והכניסי סיסמה (ברירת מחדל: שם123).
2. **פתחי את המשימה היומית** — זו הזרימה הראשית (מילים + תרגול + סיפור).
3. **קראי סיפורים נוספים** — 📚 סיפורים (מותאם אישית) או 📖 סיפורים שלמדתי (היסטוריה).
4. **עייני בכל המילים** — אופציונלי, עם סינון שיעור מתקפל.
5. **בדקי את ההתקדמות** ב"הלמידה שלי" — מילים קשות וקלות.
6. **הורים:** מצב מורה (`teacher123`) — הוספת מילים, מבחנים וניהול תלמידות.

> **סיסמת מורה (ברירת מחדל):** `teacher123`  
> **סיסמת תלמידה (ברירת מחדל):** שם + `123` — למשל `נועה123`

### הוספת מילה חדשה

| שדה | הסבר |
|-----|------|

### צפייה ומחיקה

כל המילים מוצגות **בטבלה**. למחיקה: בחרי מהרשימה ולחצי **🗑️ מחקי מילה נבחרת**.

## 🔊 הגייה (Text-to-Speech)

כפתור 🔊 זמין ב: **מילים**, **תרגול**, **סיפור**, **הלמידה שלי**.

קבצים נשמרים ב-`.cache/tts/` — לא צריך להוריד שוב מהאינטרנט.

## 💾 נתונים

כל הנתונים נשמרים **מקומית** ב-`progress.db`. אין שליחה לשרת חיצוני.

**איפוס התקדמות** (במסך "התקדמות") מוחק גם את נתוני הלמידה לכל מילה. מילות מורה **לא** נמחקות.

## 🛠️ פתרון בעיות

| בעיה | פתרון |
|------|--------|
| `command not found: streamlit` | הפעילו venv והריצו `pip install -r requirements.txt` |
| `FileNotFoundError: words.csv` | הריצו מתוך תיקיית `english_game` |
| הגייה לא עובדת | בדקו אינטרנט; מחקו `.cache/tts/` ונסו שוב |
| רוצים לאפס התקדמות | "התקדמות" → "אפסי התקדמות" |
